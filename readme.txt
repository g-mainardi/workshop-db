DB-Workshop
Applicazione web per la gestione di un'autofficina.

Generazione DB
Per la generazione della struttura del database è necessario utilizzare il file 'officina.sql' (presente nella cartella 'database') che contiene le ddl relative alla base dati.

Esecuzione dell'Applicazione
Per eseguire l'applicazione web occorre caricare in locale la cartella 'app-web' contentene i file di progetto e poi si 
potrà accedere alla pagina iniziale del sistema attraverso il localhost nella porta selezionata.
